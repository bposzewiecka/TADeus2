Genome browser

Usage:

  • Click to zoom in
  • Click to zoom out
  • Click to decrease start and end coordinates by 50% of current plot width
  • Click to decrease start and end coordinates by 25% of current plot width
  • Click to increase start and end coordinates by 25% of current plot width
  • Click to increase start and end coordinates by 50% of current plot width
  • Click to go to specified genomic location specified in text input
  • Click to filter BED data by name (if name filter of at least one track is set to 'Yes')
  • Click to edit the plot
