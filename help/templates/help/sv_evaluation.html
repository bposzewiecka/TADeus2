Structural Variants evaluation

To evaluate structural variant select "Evalute SVs" menu option.

Entering coordinates of breakpoints of structural variants

Input data needed for the evaluation are simply coordinates of structural variant.

User can provide input data using following methods:

  • Enter coordinates of breakpoints using form
  • Enter list of coordinates in BED format in text area
  • Upload file containing list of coordinates in BED format

Entering coordinates of breakpoints using form

After selecting "Evalute SVs" menu option click "Create" button.

Then fill the form with name of evaluation.

{% include 'help/figure.html' with title='"Evaluation of SVs" form' img_src='help/sv-eval-form.png' no='1' %}

Click "Add new breakpoint" button and fill the form with:

  • Name
  • Chromosome
  • Start coordinate
  • End coordinte

of the breakpoint.

{% include 'help/figure.html' with title='"New breakpoint of structural variant" form' img_src='help/sv-eval-coord-form.png' no='2' %}

Entering list of coordinates in BED format using text area

After selecting "Evalute SVs" menu option click "Create by direct input" button.

Then fill the form with the name of evaluation and paste in the coordinates of breakpoints in BED format. Click "Create" button.

{% include 'help/figure.html' with title='"Evaluation of SVs" form' img_src='help/sv-eval-paste.png' no='3' %}

Uploading file containing list of coordinates in BED format

After selecting "Evalute SVs" menu option click "Create by file upoad" button.

Then fill the form with name of evaluation and upload file in BED format. Click "Create" button.
