Web server implementation

TADeus is freely available at http://bioputer.mimuw.edu.pl/tadeus and is intended to be systematically maintained.

TADeus is implemented as a Python/Django web application using MySQL database as data storage. The source code of this application is deposited on Github (https://github.com/bposzewiecka/TADeus2).

TADeus is distributed under Gnu General Public Licence v3.0.

Fragments of code from HiCExplorer are reused in the track plot module.

TADeus contains information on the public datasources used and links to websites from they were obtained (http://bioputer.mimuw.edu.pl:82/datasources/).
